Principais formatações

Negrito / Destaque

Neste frase temos um termo em negrito usando a tab B (não semântica).

Nesta frase, temos um termo em destaque usando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântico)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro parágrago, temos outro texto marcado no final.

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Com as TAGs BIG e SMALL.

Texto deletado

Podemos marcar um texto como cacelado para indicar que ele deve ser lido, mas não considerado com a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase para indicar que ele foi adicionado depois com a tag INS.

Existe também o sublinhado com a tag U.

Texto sobrescrito

Para indicar coisas do tipo do tipo x20+3. Com a tag SUP.

Texto subscrito

Você pode inserir coisas do tipo H2O. Com a tag SUB.
